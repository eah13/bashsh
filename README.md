bashsh
======

The basics of bash scripting

## Usage
```
git clone https://github.com/eah13/bashsh.git
cd bashsh
source bash.sh
```
The script will take it from there.

## Test Yourself
When you're ready, fill in the blanks on yourscript.sh:
```
nano yourscript.sh
```
